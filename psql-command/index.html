Essential commands for Database Administrators

Connection
CLI
psql -U postgres -d mydb -h localhost -p 5432
Full connection with host/port
psql "postgresql://user:pass@host:5432/dbname"
Connection string URI format
psql -U postgres -d mydb -W
Force password prompt
psql -U postgres -c "SELECT version();"
Run single command and exit
\conninfo — show current connection info inside psql

Meta Commands
\cmd
Navigation
\l or \l+
List databases (+ = verbose)
\c dbname
Switch database
\dt \dt+ \dt schema.*
List tables (verbose / by schema)
\d tablename
Describe table structure
\di \dv \ds \df
Indexes / Views / Sequences / Functions
Output
\x
Toggle expanded (vertical) output
\timing
Toggle query execution timing
\o /tmp/out.txt
Send output to file
\i /path/to/file.sql
Execute SQL file
\e
Open last query in $EDITOR
\q
Quit psql

Database Management
DDL

CREATE DATABASE mydb_clone TEMPLATE mydb;
Clone database from template
DROP DATABASE mydb;
Drop database (irreversible!)
ALTER DATABASE mydb RENAME TO newname;
Rename database
SELECT pg_size_pretty(pg_database_size('mydb'));
Get database size
SELECT datname, pg_size_pretty(pg_database_size(datname)) FROM pg_database ORDER BY pg_database_size(datname) DESC;
All databases with sizes

Active Sessions
pg_stat
SELECT pid, usename, datname, state, query_start, left(query,80) FROM pg_stat_activity WHERE state != 'idle';
Active (non-idle) queries
SELECT pid, now() - pg_stat_activity.query_start AS duration, query FROM pg_stat_activity WHERE state = 'active' ORDER BY duration DESC;
Queries sorted by duration
SELECT count(*), datname FROM pg_stat_activity GROUP BY datname;
Connection count per database
SELECT pg_cancel_backend(pid);
Cancel a query gracefully
SELECT pg_terminate_backend(pid);
Kill a backend process hard
⚠ pg_terminate_backend terminates the entire connection — use pg_cancel_backend first.

Locks & Blocking
Concurrency
SELECT pid, relname, mode, granted FROM pg_locks JOIN pg_class ON pg_locks.relation = pg_class.oid WHERE relname NOT LIKE 'pg_%';
All table locks
SELECT blocked.pid, blocked.query, blocking.pid AS blocking_pid, blocking.query AS blocking_query FROM pg_stat_activity blocked JOIN pg_stat_activity blocking ON blocking.pid = ANY(pg_blocking_pids(blocked.pid)) WHERE cardinality(pg_blocking_pids(blocked.pid)) > 0;
Blocked queries + who's blocking
SELECT pid, pg_blocking_pids(pid) AS blocked_by FROM pg_stat_activity WHERE cardinality(pg_blocking_pids(pid)) > 0;
Quick view: who is blocked by whom
💾
Backup & Restore
pg_dump
Backup
pg_dump -U postgres -d mydb -Fc -f mydb.dump
Custom compressed format (recommended)
pg_dump -U postgres -d mydb -Fp -f mydb.sql
Plain SQL format
pg_dumpall -U postgres --globals-only -f globals.sql
Backup roles & tablespaces only
Restore
pg_restore -U postgres -d mydb -Fc mydb.dump
Restore from custom format
pg_restore -U postgres -d mydb --table=orders mydb.dump
Restore single table only
psql -U postgres -d mydb -f mydb.sql
Restore from plain SQL

Performance & EXPLAIN
Query Tuning
EXPLAIN SELECT * FROM orders WHERE id = 1;
Show query plan (no execution)
EXPLAIN (ANALYZE, BUFFERS, FORMAT JSON) SELECT ...;
Full execution + buffer stats
SELECT schemaname, tablename, pg_size_pretty(pg_total_relation_size(schemaname||'.'||tablename)) AS size FROM pg_tables ORDER BY pg_total_relation_size(schemaname||'.'||tablename) DESC LIMIT 20;
Top 20 largest tables
SELECT relname, seq_scan, idx_scan, n_live_tup FROM pg_stat_user_tables ORDER BY seq_scan DESC LIMIT 10;
Tables with most sequential scans (missing indexes)
SELECT indexname, idx_scan, idx_tup_read, idx_tup_fetch FROM pg_stat_user_indexes ORDER BY idx_scan DESC;
Index usage stats
SELECT query, calls, total_exec_time, mean_exec_time, rows FROM pg_stat_statements ORDER BY mean_exec_time DESC LIMIT 10;
Slowest queries (requires pg_stat_statements)

VACUUM & Maintenance
Health
VACUUM tablename;
Reclaim dead tuple space
VACUUM FULL tablename;
Rewrite table (locks!) — free disk
VACUUM ANALYZE tablename;
Vacuum + update planner stats
ANALYZE tablename;
Update statistics only
REINDEX TABLE tablename;
Rebuild all indexes on table
REINDEX INDEX CONCURRENTLY idx_name;
Rebuild index without locking (PG 12+)
SELECT relname, n_dead_tup, last_vacuum, last_autovacuum FROM pg_stat_user_tables ORDER BY n_dead_tup DESC;
Tables needing vacuum (dead tuples)

Table & Schema DDL
Structure
CREATE TABLE orders (id SERIAL PRIMARY KEY, amount NUMERIC(10,2), created_at TIMESTAMPTZ DEFAULT NOW());
Create table with common types
ALTER TABLE orders ADD COLUMN status TEXT DEFAULT 'pending';
Add column with default
ALTER TABLE orders ALTER COLUMN amount SET NOT NULL;
Add NOT NULL constraint
CREATE INDEX CONCURRENTLY idx_orders_status ON orders (status);
Non-locking index creation
CREATE SCHEMA reporting AUTHORIZATION appuser;
Create schema with owner
SET search_path TO reporting, public;
Set schema search path

Data Import / Export
COPY
COPY orders FROM '/tmp/orders.csv' CSV HEADER;
Import CSV (server-side path)
\copy orders FROM '/local/orders.csv' CSV HEADER;
Import CSV (client-side — no superuser needed)
COPY orders TO '/tmp/orders_export.csv' CSV HEADER;
Export table to CSV
\copy (SELECT * FROM orders WHERE status='done') TO '/tmp/done.csv' CSV HEADER;
Export query result to CSV
COPY orders FROM STDIN WITH (FORMAT CSV, DELIMITER '|', NULL '\N');
Pipe-delimited with custom NULL

Replication & WAL
HA
SELECT * FROM pg_stat_replication;
Connected standbys + replication lag
SELECT pg_is_in_recovery();
Is this node a standby? (t/f)
SELECT now() - pg_last_xact_replay_timestamp() AS lag;
Replication lag on standby
SELECT slot_name, active, pg_size_pretty(pg_wal_lsn_diff(pg_current_wal_lsn(), restart_lsn)) AS retained FROM pg_replication_slots;
Replication slots + WAL retained
SELECT pg_walfile_name(pg_current_wal_lsn());
Current WAL file name
⚠ Inactive replication slots retain WAL indefinitely — monitor and drop stale slots.

Config & Runtime
postgresql.conf
SHOW max_connections; -- any param
Show current parameter value
SHOW config_file;
Path to postgresql.conf
SELECT name, setting, unit, source FROM pg_settings WHERE source = 'configuration file';
Non-default config values
ALTER SYSTEM SET work_mem = '64MB';
Change setting (writes postgresql.auto.conf)
SELECT pg_reload_conf();
Reload config without restart
SELECT name, setting FROM pg_settings WHERE pending_restart = true;
Params pending restart
SELECT pg_postmaster_start_time();
Server start time (uptime reference)
